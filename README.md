## About Laravel-2FA-Integration

Connect Google Authenticator 2FA in Laravel login

- Connect Google Authenticator for each user
- 2FA code validator for each user in login

## Steps to Setup:
- composer i
- php artisan migrate
- php artisan storage:link
- npm run i
- npm run build
